# Cloud-Based-Intrusion-Detection-Approach-Using-Machine-Learning-Techniques.

How to Run

# Step 1 — Install Dependencies
pip install scikit-learn pandas numpy dash plotly

# Step 2 — Run the Pipeline
python IDS_Complete.py

# Step 3 — Open the Dashboard
Once you see "Dash is running on http://127.0.0.1:8050/" open your browser and go to:
http://127.0.0.1:8050

---

# Dashboard Features

- KPI Cards — Total events, attacks detected, detection rate, overall accuracy
- Attack Distribution Pie Chart
- Confusion Matrix Heatmap
- Detection Rate by Attack Type
- Feature Importance Chart

---

# Tech Stack

- Python 3.x
- scikit-learn — Isolation Forest, Random Forest
- pandas / numpy — Data processing
- Dash + Plotly — Interactive dashboard
